# Assignment Grading Interface

Standalone copy of the grading platform from the main assignment repo.

## Quick start

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python grading_app/app.py
```

Open `http://127.0.0.1:5000`

See `GRADING_INTERFACE_GUIDE.md` for full documentation.
